# Analista de Vício em Telas

Projeto desenvolvido para o curso **IA e Soft Skills para Programadores - Online - 3.0** da **Geração Tech**. O código é baseado em projeto de exemplo adquirido ao longo do curso.

# Objetivo

Aplicar os conhecimentos aprendidos ao longo do curso, criando um agente IA abordando um tema específico. O tema é sobre vício em telas, onde o agente IA pega informações de hábito de uso e analisa se o usuário possui vício em telas e como tratar isso.

# Como rodar o projeto

1. Clonar o repositório:
   ```bash
   git clone https://github.com/JanilsonFerreira/projeto-vicio-tela.git
   ```

2. Instalar as dependências do Node.js dentro das pastas `frontend` e `backend`:
   ```bash
   npm install
   ```

3. Adquirir uma chave API gratuita (ou paga) do Gemini AI no [Google AI Studio](https://aistudio.google.com/app/apikey) e colar ela na seguinte linha do arquivo `.env` na pasta `backend`:
   ```bash
   GEMINI_API_KEY=sua_chave_api_aqui
   ```

4. Emular o projeto localmente rodando o seguinte comando dentro das pastas `frontend` e `backend`:
   ```bash
   npm run dev
   ```
   
5. Acessar a seguinte página em seu navegador de preferência:
   ```
   http://localhost:3000/
   ```
